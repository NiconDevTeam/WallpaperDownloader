# Welcome to Wallpaper Downloader Repository #

## Information ##
This repository contains all the source code related to Wallpaper Downloader application. It is totally free, open source and you can use it or fork it if you wish. Please, let me know if you are interested in doing this.

## Quick summary ##
This is a JAVA open source software project developed for downloading wallpapers from different sources. It is a crossed-platform standalone application.
